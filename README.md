# homebrew-tap

[ieee0824](https://github.com/ieee0824) の Homebrew tap リポジトリです。

## tap の追加

```sh
brew tap ieee0824/tap
```

## フォーミュラ一覧

### apricot

Apple Container 向けの Docker Compose 代替ツールです。

- **対応環境**: macOS (Apple Silicon)
- **リポジトリ**: https://github.com/ieee0824/apricot

```sh
brew install ieee0824/tap/apricot
```

### vpp2txt

Voice Peak (.vpp) プロジェクトファイルをテキストに変換するツールです。

- **対応環境**: macOS / Linux (ARM64 / AMD64)
- **リポジトリ**: https://github.com/ieee0824/vpp2txt

```sh
brew install ieee0824/tap/vpp2txt
```
